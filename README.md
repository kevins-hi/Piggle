# Piggle

Piggle is a fun iOS app to mess around on. Its most complete feature is a weather app you can access by clicking the weather button in the top right corner. Among it's other features are a (currently) non-functional search bar, toggleable background music, and a few hidden easter eggs!

How to get Piggle on your iPhone:
1. Install Xcode on your laptop
2. Download the code from this repository
3. Open up the .xcodeproj file in Xcode
4. Connect your iPhone to your laptop with an adapter cable
5. Click Product > Build in the menu bar
6. In the menu bar, go to Window > Devices and Simulators
7. Drag Piggle.app from the Xcode file tree into Installed Apps
